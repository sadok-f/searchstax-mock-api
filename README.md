
Simulate SearchStax API response.
Based on https://www.searchstax.com/docs/searchstax-cloud-deployment-api/


## Install

```bash
git clone github.com/sadok-f/searchstax-mock-api
cd searchstax-mock-api/
go install
```


## Serve Mock Server
```bash
./searchstax-mock-api
```
